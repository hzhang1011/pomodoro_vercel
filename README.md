# 番茄钟（Vercel 一键部署版）

一个轻量、无需后端的番茄钟（Pomodoro）网页应用。把本仓库推到 GitHub，然后用 Vercel 一键部署即可上线。

## 功能
- 专注 / 短休息 / 长休息
- 自定义时长（分钟）
- 自动开始下个阶段、结束提示音
- 本地保存设置（localStorage）
- 导出/导入设置 JSON
- 键盘快捷键：空格 开始/暂停，R 重置

## 快速部署（一步）
1. 新建一个 GitHub 仓库（Public 或 Private）。
2. 将本仓库的文件（`index.html`、`README.md`）推到该仓库的 `main` 分支。
